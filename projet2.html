Projet : PlantAxis


Description :


Ce projet est une création d'un site internet d’horticulture pour but d'élargir mes connaissances dans les langages du web et de m'améliorer dans le côté Webdesign. Dans un premier temps, j'ai commencé à réaliser ce site via Wix qui est un logiciel que je connaissais depuis quelques années. Il sera amélioré et traduit sous Worldpress, car c'est un outil grandement demandé dans le marché du web et sera l'équivalent d'une maquette. Enfin je le traduirais en Html, Css et JavaScript, hébergé sous GitHub pour finaliser ce projet.
